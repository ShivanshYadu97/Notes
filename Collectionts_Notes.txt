=> Traversals -

	NOTE - List can be traversed using 4 types- ListIterator, Iterator, Iterable or for-each & forEach method.
	
	1. ArrayList - cause it is the part of List so it can also be get traversed using 3 types.
	
				1. for Each loop - Forward Traversal
				
								 => ArrayList<String> names = new ArrayList<String> ();
									names.add("Ramesh");
									names.add("Suresh");
									names.add("Riya");
									names.add("Piya");
									
									for(String str:names){
										syso(str);			//Forward traversal
										StringBuffer br = new StringBuffer(str);
										syso(br.reverse());						//each word will get reverse but the order will be forward
									}
									
				2. Iterator - Forward Traversal
				
							=> Iterator<String> itr = names.iterator();
							   
							   while(itr.hasNext()){
									String next = itr.next();
									syso(next);					//Forward traversal
							   }
							   
				3. ListIterator - Backward Traversal
								NOTE - listIterator extends the Iterator.
				
								=> ListIterator<String> ls = names.listIterator(names.size());		//cursor needs to start from backside so thats why we provided size here
								   
								   while(ls.hasPrevious()){
										String previous = ls.previous();
										syso(previous);					//Backward traversal
								   }
								   
				NOTE- Enumeration -  it can be used inside Vector only.
				
				
				4. forEach method - Forward Traversal
				
								  NOTE - here we provide the Consumer to the forEach cause the Consumer is a Functional Interface so thats why we had to use lambdas expression here.
								  
								  => names.forEach(e->{
										syso(e);		//Forward Traversal
									})
									
				
	NOTE- Set can be traversed using 2 types
					
	2. TreeSet- cause it is the part of Set so the TreeSet can be traversed using 2 types.
	
				1. forEach method - Forward Traversal
				
								  => TreeSet<String> set = new TreeSet<>();
									 set.addAll(names);
									 
									 set.forEach(e->{
										syso(e);
									 })
									 
									 NOTE - using Comparator & Comparable we provide the custom sorting logic. Suppose float,int and for Double values.
									 
				2. Iterator - Forward Traversal
				
							=> TreeSet<String> ts = new TreeSet<>();
							   ts.addAll(names);
							   
							   Iterator<Integer> it = ts.iterator();
							   
							   while(it.hasNext()){
									syso(it.next());
							   }
				
	3. HashMap - 
	
				1. forEach method - 
								  => HashMap<String,Integer> price = new HashMap<>();
								     price.put("Potato",20);

									 price.forEach((e1,e2)->{		//here e1-key, e2- value
